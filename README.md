# 🚨 AI Helmet Violation Monitoring System

An end-to-end AI-powered Computer Vision application that automatically detects whether a motorcycle rider is wearing a helmet using a custom-trained **YOLOv11** model.

The project supports **Image Detection**, **Video Detection**, and **Real-Time Webcam Detection**, providing a practical demonstration of how AI can be applied to intelligent traffic monitoring systems.

# 📌 Project Overview

Road safety remains a major concern, and helmet compliance is one of the most important factors in reducing serious injuries during road accidents.

This project demonstrates how Computer Vision can automate helmet detection and simulate a real-world helmet violation monitoring workflow.

Instead of building only an object detection model, the project focuses on creating a complete AI application—from dataset preparation to deployment.

# 🚀 How to Run Locally

### Clone Repository

```bash
git clone https://github.com/ashu2212-debug/AI-Helmet-Violation-Monitoring-System.git
```

```bash
cd AI-Helmet-Violation-Monitoring-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch Application

```bash
python app.py
```

The Gradio interface will open automatically.
